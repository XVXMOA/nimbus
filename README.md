# Nimbus Keyboards

A premium keyboard showcase website built with Next.js, featuring 3D visualizations and modern web technologies.

**Made by Saleh** - This project showcases custom mechanical keyboards with interactive 3D models and a sleek, modern interface.

## Features

- 🎨 Modern, responsive design with Tailwind CSS
- 🎮 Interactive 3D keyboard models using Three.js and React Three Fiber
- ⚡ Smooth animations with GSAP
- 📱 Fully responsive design for all devices
- 🎭 Beautiful UI components with Radix UI
- 📝 Content management with Prismic CMS

## Technologies Used

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Three.js** - 3D graphics
- **React Three Fiber** - React renderer for Three.js
- **GSAP** - Animation library
- **Prismic** - Headless CMS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Project Structure

- `src/app/` - Next.js app router pages and API routes
- `src/components/` - Reusable React components
- `src/slices/` - Prismic slice components
- `src/lib/` - Utility functions and configurations
